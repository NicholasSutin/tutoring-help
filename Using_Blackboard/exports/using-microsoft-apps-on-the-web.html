Using Microsoft Apps on the web

This will assume you know your college email and password.

Go to Outlook and sign in

  • Click on the waffle in the top left
  • You should see several apps.
    • Word for typing documents (PDFs, DOCXs)
    • Excel for spreadsheets
    • PowerPoint for creating presentationsa
  • Click to see Google Alternatives
    • docs.google.com instead of Word (slightly simpler, allows PDF and DOCX exports)
    • sheets.google.com instead of Excel. slightly less functionality, easier for beginners.
    • slides.google.com much easier to use and navigate, allows .PPT exports.

Using Word

The file opening process and saving process is the same as Excel and Powerpoint

  • To edit a document you downloaded, select “Upload a file”
  • a new window with “File Explorer” will open.
    1. Select which folder or location the file is in
    1. Click the file name once
    1. select “Open”
  • You will then be able to edit the document

Saving a document

Saving as a PDF (simplest):

  • go to the top left: File → Export → Download as PDF

Saving as a DOCX:

  • File → Create a Copy → Download Copy
  • A popup may appear, select “Download a Copy”
  • You will be taken to a new tab, which should open File Explorer. Select where you will download (usually the Downloads folder), and click “Save.”
