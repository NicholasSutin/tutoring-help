Using Word and saving Word Documents to files

If Microsoft Word is already installed

If you need to use it in a web browser

Click the  button, and type “Word”

Open the App.

Go first to Outlook, and sign in with your student account.

Go to the top left, where the 9 squares or “waffle” is, and search for “Word”

Create a blank document, type [CTRL, A], and choose the font you need. If you have a language-related class, it will often be “Times New Roman,” but make sure to read the assignment instructions and syllabus so you know how to format your documents.

You can change the font as well.

To Save and Export your file

Click “File” on the top left, then click “Save As”

Type in the box here to change the title

Select the file type. Usually, professors like the format in either “Word Document (*.docx)”, or in “PDF (*.pdf)”
